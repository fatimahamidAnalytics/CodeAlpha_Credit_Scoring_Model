# Credit Scoring Model

Predicts individual creditworthiness using past financial data.

## Models Used
- Logistic Regression (ROC-AUC: 0.714)
- Decision Tree (ROC-AUC: 0.848)
- Random Forest (ROC-AUC: 0.846)

## Dataset
Give Me Some Credit - Kaggle

## Libraries
pandas, numpy, scikit-learn, matplotlib, seaborn
